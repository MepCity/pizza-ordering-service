# Pizza Ordering Service

Minimal pizza siparis servisi. Projenin amaci, kucuk bir FastAPI uygulamasi etrafinda test, CI/CD, container, Kubernetes, monitoring ve performans altyapisini kurmaktir.

## Kapsam

- Menu listeleme
- Siparis olusturma
- Siparis listeleme
- Siparis detayi goruntuleme
- Siparis durumu guncelleme
- Kupon uygulama
- Siparis ozetini S3'e yazma

## Teknolojiler

- FastAPI
- PostgreSQL
- SQLAlchemy
- LocalStack S3
- Pytest
- Playwright
- k6
- Prometheus + Grafana

## Durum

Ilk proje iskeleti olusturuluyor.
